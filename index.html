📋 What is This?

DOCUGRAPH CNN-Only is a baseline document analysis system that uses pure Convolutional Neural Networks (CNN) without Graph Neural Network components. It provides:

  • Layout Detection: Identifies document structure and regions
  • Element Classification: Categorizes headers, text, tables, and figures
  • Bounding Box Generation: Precise region localization
  • Confidence Scoring: Reliability metrics for each prediction

🚀 Getting Started

1. Click "Start Analysis" to open the analysis interface

2. Upload a document image or PDF

3. Click "Analyze Layout" to process

4. View results with visualized bounding boxes
