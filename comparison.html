Key Insights

🎯 Accuracy Improvement

CNN+GNN improves accuracy by 27% over CNN-only through graph relationship analysis. This is critical for complex document layouts.

🧠 Context Understanding

CNN-only processes regions independently. CNN+GNN understands relationships between regions, leading to better hierarchical analysis.

⚡ Speed vs Accuracy Trade-off

CNN-only is ~1.8x faster but CNN+GNN provides superior accuracy. Choose based on your latency requirements.

📊 Table Detection

CNN-only detects table regions. CNN+GNN reconstructs actual table grids with proper cell alignment through relational reasoning.

📈 Complex Documents

On complex documents with multiple layouts, CNN+GNN shows the most improvement as graph relationships become more important.

💼 Use Cases

CNN-only: Real-time or simple documents. CNN+GNN: High-accuracy production systems, research papers, complex forms.

✓ Recommendation

For production document analysis with high accuracy requirements, DOCUGRAPH's CNN+GNN architecture is recommended. It provides 27% better accuracy than CNN-only while maintaining reasonable processing times. CNN-only is valuable as a fast baseline for comparison purposes or when processing latency is critical.
